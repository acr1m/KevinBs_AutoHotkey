regex_convertText_SciTE_API()

Converts whitespace for .api files used by SciTE and removes a limited amount of comment delimiters. 

regex_convertText_SciTE_API(p_replaceSelection)

Parameters

p_replaceSelection
BOOLEAN (Default := false)

Returns

STRING - returns selected text in a format style that is friendly for the user.api.ahk file.

Remarks

This is used to manually copy and paste from file to file.
